Prem Oracle — Support

How do I play?

Pick the final score for any Premier League fixture using the +/− buttons and tap "Lock it in". You can change a pick any time before kick-off. Scoring: exact score 5 points; correct draw with wrong score 2; correct winner and goal difference 2; correct winner only 1.

How do leagues work?

In the League tab, create a league and share the code with friends. Everyone's picks stay hidden until kick-off, then reveal at once. The table updates as results come in.

I got a new phone — how do I restore my picks?

Your three-word recovery code is shown in the League tab. On the new device, open League → enter the code → Restore.

A match was postponed — what happens to my pick?

Postponed or abandoned matches are void (0 points for everyone) until they are rescheduled, at which point predictions reopen.

Something's not working

Email adambigwood@me.com and include your recovery code if the issue involves your picks or league.
